API para desarrolladores

Captura y aportes

Recibe archivos, páginas, notas y videos. Cada aporte queda listo para revisión y publicación.

Gestor autenticado: se usará tu correo guardado para las subidas y no tendrás que escribirlo en cada video.

Pega el embed completo. El sistema resolverá la URL original, metadata y visualización reutilizable.

Búsqueda semántica y alternativas

Encuentra contenido relacionado, respuestas similares y material útil para ofrecer alternativas.

Catálogo comercial

Espacio dedicado a vender contenido, ofrecer opciones y filtrar alternativas por plataforma, categoría o tendencia.

Este módulo solo está disponible para vendedores activos asignados a una empresa.

Biblioteca comercial para vendedores

Operación y moderación

    Categorías jerárquicas

    Crea ramas y subramas ilimitadas. Ejemplo: Películas > Romántica > Romance asiático.

    Las propuestas hechas por usuarios quedan pendientes hasta que un administrador las apruebe o reubique el contenido en otra rama.

      Plataformas y marcas

      Administra marcas como Netflix, Prime Video, Amazon o Movistar TV para filtrar el catálogo comercial.

        Empresas

        Crea la empresa y luego asigna los perfiles vendedor que podrán entrar al módulo comercial.

          Perfiles de acceso

          Crea vendedores y gestores de contenido. El vendedor debe pertenecer a una empresa; el gestor puede pertenecer a una empresa o quedar solo bajo administración.
